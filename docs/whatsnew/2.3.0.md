# What's New in 2.3.0 ✨

## 🚀 Instant Application Discovery
We've completely overhauled the **Command Palette (Ctrl+K)**. You no longer need to type a query to see your apps. As soon as you open the palette, you'll see a complete, organized list of everything available to you.
- **Recently Launched**: Your most used apps are now pinned to the top.
- **Alphabetical Ordering**: Finding apps within categories is now faster than ever.

## ⚡ Direct SSO Access
Need to get in fast? We've added a new `/sso` short-link.
- **One-Click Login**: Automatically triggers the Keycloak authentication flow.
- **Bookmarkable**: Add `https://techhub.example.com/sso` to your browser bookmarks for the fastest possible entry.

## 🛡️ Security Hardening
We've patched several core dependencies to protect against emerging vulnerabilities:
- **XSS Protection**: Updated PostCSS and SVG sanitization logic.
- **Memory Safety**: Hardened the `uuid` buffer handling.
- **Technical Debt**: Conducted a full audit of the administrative codebase to remove dead code and optimize performance.

## 📚 Documentation 2.0
We've expanded our technical documentation to include:
- **Full API Reference**: Every endpoint, documented.
- **Architecture Deep Dive**: Updated diagrams and lifecycle flows.
- **Security Whitepaper**: Detailed breakdown of our "Defense-in-Depth" strategy.
