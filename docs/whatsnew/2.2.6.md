# What's New in 2.2.6 🛠️

## ⏱️ Reliable Session Management
We've fixed a critical bug where the **Idle Timeout** would reset prematurely due to background session refreshes.
- **Accurate Enforcements**: The 20-minute idle timer and 8-hour absolute timer now fire precisely as configured.
- **Client-Server Alignment**: Improved synchronization between the browser session guard and the server-side proxy middleware.

## 🛡️ Concurrent Session Control
Manage your active logins with ease:
- **Session Heartbeats**: We now track active browser tabs in real-time.
- **Themed Banners**: Receive beautiful, glassmorphic notifications if your account is logged in elsewhere.
- **Remote Sign-out**: Terminate all other sessions with a single click.

## 🐛 Bug Fixes & Stability
- Resolved "Missing Regex Anchors" in E2E tests to prevent URL spoofing matches.
- Cleaned up over 70 unused variables and imports identified by CodeQL.
- Fixed `bcryptjs` and `ioredis` inclusion in standalone Docker builds.
